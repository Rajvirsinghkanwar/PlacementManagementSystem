# Placement Management System

## Overview

The Placement Management System is a web-based application developed using **Django (Python)** that helps automate and manage the campus placement process. It allows students to register, log in, and participate in recruitment activities while administrators manage student records and placement opportunities.

## Features

* Student Registration
* Student Login
* Admin Login
* Manage Student Data
* Placement Information Management

## Technologies Used

* Python
* Django Framework
* HTML
* CSS
* SQLite Database

## How to Run the Project

1. Clone the repository
2. Navigate to the project folder
3. Run the server

```
python manage.py runserver
```

4. Open browser and go to:

```
http://127.0.0.1:8000/
```
